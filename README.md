## Projeto DevOps

Landing page da Metro Financial desenvolvida como projeto prático para a disciplina de DevOps. O projeto demonstra a implementação de uma aplicação web full-stack com frontend React, backend Node.js e banco de dados MySQL, preparada para práticas de CI/CD, containerização e monitoramento.

## Descrição do Projeto

A Metro Financial é uma plataforma bancária digital fictícia que oferece serviços financeiros modernos. O projeto consiste em uma landing page responsiva e interativa que apresenta os serviços da empresa, sua história e missão, além de um formulário de contato funcional integrado a um banco de dados.

O objetivo principal é criar uma aplicação completa que sirva como base para a aplicação de conceitos de DevOps, incluindo integração contínua, entrega contínua, containerização com Docker, orquestração, monitoramento de infraestrutura e análise de performance.

## Tecnologias Utilizadas

- React + Vite
- Tailwind CSS
- Node.js + Express
- MySQL
- Lucide React (ícones)

## Funcionalidades

- Design responsivo e moderno
- Seção de serviços com 3 cards
- Seção "Sobre Nós" com história e missão
- Formulário de contato funcional
- Integração com banco de dados MySQL
- Footer com links sociais
